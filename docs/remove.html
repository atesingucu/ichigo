SİLME TÜRKÇE
  • /docs/ dizini hariç tüm dosyaları çıkarttığınız FTP yazılımı ile sitenize giriniz.
  • Sonra temanın ana dizininde, klavyeden CTRL+A (Mac OS X için, CMD+A) düğmesine basınız.
  • Son olarak DELETE (Mac OS X için, CMD+Backspace) düğmesine basınız.
REMOVING INSTRUCTIONS
  • Using the FTP software with which you extracted the files, access your site excluding the /docs/ directory.
  • Then, in the main directory of the theme, press CTRL+A (CMD+A on Mac OS X) on your keyboard.
  • Finally, press the DELETE (or CMD+Backspace on Mac OS X) button.
削除作業
  • /docs/ ディレクトリを除く全てのファイルを展開したFTPソフトウェアを使って、サイトにアクセスしてください。
  • 次に、テーマのメインディレクトリで、キーボードから CTRL+A (Mac OS XではCMD+A) を押してください。
  • 最後に、DELETE (Mac OS XではCMD+Backspace) ボタンを押してください。
