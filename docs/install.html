KURULUM TÜRKÇE
  • Temayı sitemizden indiriniz.
  • Bir dizine çıkartınız.
  • /docs/ dizini hariç tüm dosyaları bir FTP yazılımı ile sitenize yükleyiniz.
INSTALLATION IN ENGLISH
  • Download the theme from our website.
  • Extract it to a directory.
  • Upload all files, except the /docs/ directory, to your site using FTP software.
日本語でのインストール
  • 当社のウェブサイトからテーマをダウンロードしてください。
  • ディレクトリに展開してください。
  • /docs/ ディレクトリを除くすべてのファイルをFTPソフトウェアを使用してサイトにアップロードしてください。
